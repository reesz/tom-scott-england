# Graphics Overhaul Design

**Date:** 2026-04-09
**Status:** Draft

## Overview

Replace the current raw WebGL procedural background with a Three.js-powered rendering pipeline featuring real DEM terrain data, stylized modern water shaders, full British Isles geographic context, and multi-pass post-processing. The goal is an awwwards-worthy visual experience with 2.5D parallax depth.

## Problems Being Solved

1. **Terrain heightmap not land-masked** — procedural noise renders across the entire canvas (land + ocean) with no relation to real English topography.
2. **Water animation is low quality** — two simplex noise layers + a sin stipple. Reads as noisy stripes, not water.
3. **No full island outline** — GeoJSON only has England's 47 counties. Great Britain floats in a void without Scotland, Wales, or surrounding geography.
4. **Overall visual quality** — needs to reach awwwards-level polish.

## Design Decisions

| Decision | Choice | Alternatives Considered |
|---|---|---|
| Terrain data | Real DEM (SRTM/ETOPO) heightmap texture | Procedural with biases, hybrid DEM+procedural |
| Water style | Stylized modern (clean ripples, gradient, translucent) | Realistic ocean, antique cartographic |
| Geographic scope | Full British Isles (GB + Ireland + IoM) | GB only, GB + near continent |
| Performance budget | Go all out (WebGL2, multi-pass, post-processing) | Balanced, progressive enhancement |
| Terrain presentation | 2.5D — flat map with elevation-based parallax | Flat shaded relief, full 3D mesh |
| Rendering library | Three.js + postprocessing | Raw WebGL2, Deck.gl/MapLibre |

## Data Assets

### DEM Heightmap Texture (~1-2MB PNG)
- Coverage: British Isles bounding box (~49N-61N, 11W-2E)
- Source: SRTM 90m or ETOPO1, processed to equirectangular PNG
- Resolution: 2048x2048
- Encoding: grayscale, black=sea level, white=max elevation (~1345m)

### Land Mask Texture (~200KB PNG)
- Rasterized from British Isles coastline (Natural Earth or OSM)
- Includes: Great Britain, Ireland, Isle of Man, Channel Islands, Isle of Wight, Hebrides, Orkney, Shetland
- White=land, Black=water
- Same resolution/projection as DEM for pixel alignment

### British Isles Outline GeoJSON (~500KB)
- Simplified coastline polygons for GB, Ireland, IoM
- Used to render muted SVG landmass shapes behind England counties
- Separate from existing `england-counties.geo.json`

All assets will be sourced and prepared during implementation. DEM data will be downloaded from a public source (e.g., SRTM via OpenTopography, ETOPO1 via NOAA, or a pre-processed British Isles DEM). The land mask will be generated by rasterizing a Natural Earth or OSM coastline shapefile to a PNG matching the DEM bounds. A Python/Node script may be written to handle the rasterization if no suitable pre-made asset exists.

## Rendering Architecture

### Three.js Scene Structure

```
Three.js Scene
├── OrthographicCamera (top-down)
├── TerrainPlane (fullscreen quad)
│   ├── Uniforms: u_dem, u_mask, u_mouse, u_time
│   └── Fragment shader:
│       ├── DEM sampling → hypsometric color mapping
│       ├── Hillshade from DEM neighbor normals
│       ├── Directional lighting (NW, cartographic convention)
│       ├── Land mask multiplication (terrain only where land=1)
│       └── Parallax: UV offset = mouseOffset * elevation * 0.015
│
├── WaterPlane (behind terrain)
│   └── Fragment shader:
│       ├── Gradient: coastal light blue → deep blue-grey
│       ├── Multi-octave animated ripples
│       ├── Shore foam at land mask transition
│       ├── Fresnel-like sheen on ripple peaks
│       └── Depth fog (darker far from land)
│
├── PaperOverlay (screen-space, rendered before post)
│   └── Subtle parchment grain + vignette, multiply blend (~0.08 opacity)
│
└── Post-Processing (EffectComposer from pmndrs/postprocessing)
    ├── Bloom (threshold 0.85, radius 0.4, strength 0.15)
    ├── Film grain (intensity 0.03, animated)
    └── Vignette (offset 0.4, darkness 0.5)
```

### Parallax Implementation
- Mouse position normalized to [-1, 1] from screen center
- Per-pixel UV offset: `mouseOffset * elevationAtPixel * 0.015`
- Mountains shift noticeably, lowlands barely move, water stays fixed
- Smooth lerp toward target at ~0.05/frame (no jitter)
- Parallax dampened during fly-to animations
- Disabled on touch devices (no mouse)

## SVG Layer Changes

### New: Background Landmasses
- Scotland, Wales, Ireland, IoM rendered as simple SVG paths behind county paths
- Fill: `rgba(180, 170, 155, 0.3)` — visible but non-interactive
- Stroke: `0.5px, rgba(90, 74, 58, 0.1)` or none
- No hover/click handlers, no labels

### Modified: England County Paths
- Remove current fill colors (green tint, hatch pattern)
- Replace with: transparent fill + stroke-only boundaries (`rgba(60, 50, 40, 0.4)`, 1px)
- Hover: subtle bright fill (`rgba(255, 255, 255, 0.15)`) + stroke brightens, 150ms ease-in
- Selected: stronger fill (`rgba(255, 255, 255, 0.2)`) + thicker stroke
- Released vs upcoming: released counties get subtle gold stroke/glow, upcoming stay neutral

### Modified: County Labels
- Keep Fraunces serif font
- Add subtle text-shadow or white halo for legibility over terrain

## Visual Specifications

### Hypsometric Color Scale
| Elevation | Color | Description |
|---|---|---|
| 0m | `#4a7c59` | Rich lowland green |
| 50m | `#6b9e5a` | Mid green |
| 150m | `#a8c36a` | Light green-yellow |
| 300m | `#c4a855` | Tan/buff |
| 500m | `#a07040` | Warm brown |
| 800m+ | `#7a5060` → light grey | Purple-brown to peaks |

### Hillshade
- Light direction: azimuth 315 (NW), altitude 45
- Ambient factor: 0.35 (shadows never full black)
- Subtle specular highlight on ridgelines

### Water Palette
| Zone | Color |
|---|---|
| Near coast | `#89b8c4` (light blue-grey) |
| Mid ocean | `#4a7a8a` (deeper teal) |
| Deep ocean (edges) | `#2a4a5a` (dark blue-grey) |
| Shore foam | `rgba(255,255,255,0.3)`, animated width pulsing |
| Ripple highlights | Subtle white specks on wave peaks |

### Post-Processing
- Bloom: threshold 0.85, radius 0.4, strength 0.15
- Grain: intensity 0.03, animated
- Vignette: offset 0.4, darkness 0.5

## useThreeScene Hook Responsibilities
- Create WebGLRenderer, Scene, OrthographicCamera
- Load DEM + land mask textures asynchronously
- Create terrain, water, paper overlay meshes with ShaderMaterial
- Set up EffectComposer with bloom, grain, vignette
- Animation loop: update u_time, u_mouse uniforms, render
- Handle resize + DPR
- Cleanup on unmount
- Respect `prefers-reduced-motion` (freeze time, center mouse)

## Mobile Fallback
- Skip bloom pass
- DPR capped at 1
- Reduce shader octave counts via uniform
- Fewer water ripple layers
- Parallax disabled (no mouse on touch devices)

## Interaction Details
- County hover: 150ms ease-in transition
- Parallax: lerp at 0.05/frame (smooth, slightly laggy follow)
- Fly-to: parallax dampened during camera animation
